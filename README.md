# CodeceptJS PoC

#### very basic utility

## Requirements:
+ **Node: https://nodejs.org/es**

## Getting Started

```bash
$ git clone https://github.com/Crowdar/LTM-codeceptjs-api-sample.git && cd "$(basename "$_" .git)"
```

### Run Tests

```bash
$ npx codeceptjs run --steps
```

### Features Builder

```bash
$ npx codeceptjs gt
```
