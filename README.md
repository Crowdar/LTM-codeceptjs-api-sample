# CodeceptJS PoC

#### very basic utility

## Requirements:
+ **Node: https://nodejs.org/es**

# Getting Started

```bash
$ git clone https://gitlab.crowdaronline.com/lippia/products/samples/codeceptjs-sample.git && cd "$(basename "$_" .git)"
```

## Run Tests

```bash
$ npx codeceptjs run --steps
```

## Features Builder

```bash
$ npx codeceptjs gt
```
